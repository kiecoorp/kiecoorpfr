On peut filtrer avec grep ou autre : {% highlight bash %} tar -tf mon_archive.tar.gz {% endhighlight %} On extrait le ou les fichiers(avec les *) avec la commande : {% highlight bash %} tar -xvf mon_archive.tar.gz "*.jpg" {% endhighlight %} ou {% highlight bash %} tar -xvf mon_archive.tar.gz "/le/bon/chemin/mon_fichier.doc" {% endhighlight %}
