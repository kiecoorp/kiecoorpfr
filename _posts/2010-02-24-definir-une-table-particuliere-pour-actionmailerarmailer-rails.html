Faisant évoluer un projet déjà existant et comportant une table email ... il m'a fallut modifier la table de base de ARMailer, rien de bien sorcier mais bon ... ça fera toujours gagner un peu de temps à certaine personne enfin j'espère :).

Les lignes de commande spéciales :

{% highlight bash %} ar_sendmail --create-migration --table-name ar_emails ar_sendmail --create-model --table-name ar_emails {% endhighlight %}

et surtout dans le environment.rb en fin de fichier.

{% highlight ruby %} require 'action_mailer/ar_mailer' ActionMailer::Base.delivery_method = :activerecord ActionMailer::ARMailer::email_class = ArEmail {% endhighlight %} et pour les commandes : {% highlight bash %} $ar_sendmail -t ArEmail --mailq {% endhighlight %}

Pour plus de compréhension, je vous incite à lire la doc  http://seattlerb.rubyforge.org/ar_mailer/ActionMailer/ARMailer.html.
